# Wei Opening Trade Skill

**版本:** 1.0.0 | **最后更新:** 2026-03-25

## 概述

**wei-opening-trade** 是一个专注于A股和港股开盘交易决策的智能分析 skill。

该 skill 结合实时股票行情数据与多模型交叉验证分析，帮助用户在以下关键时刻做出加仓决策：
- **开盘前** - 基于隔夜信息预测开盘走势并制定预案
- **交易时段** - 分析盘中动态，判断加仓时机
- **尾盘时段** - 收盘前30分钟分析尾盘加仓机会

## 核心功能

1. **智能股票识别** - 从用户输入中提取股票名称和代码
2. **实时行情获取** - 使用 MX_FinData MCP 工具获取最新行情
3. **多维度分析** - 结合技术分析、市场情绪、资金流向
4. **分时段策略** - 根据交易时间段提供针对性建议
5. **风险控制** - 内置仓位管理和止损建议

## 使用方式

本 skill 需要配合股票数据 skill 使用，先获取股票数据，再将数据作为背景信息传入查询。

**数据获取优先级：**
1. **优先使用 `/mx_finance_data` skill** - 东方财富数据源，数据全面，覆盖A港美市场
2. **备选使用 `/akshare` skill** - 开源金融数据接口，A股数据丰富

1. **根据当前时段，使用 `/mx_finance_data` skill（或 `/akshare`）获取对应维度的数据：**

2. **将获取到的股票数据作为背景信息，传入 opening trade skill：**

## 分析流程

### Step 1: 获取股票数据（前置步骤，由调用方完成）

**使用 `/mx_finance_data` skill 或 `/akshare` skill 获取实时行情数据，根据时段选择不同的数据维度：**

#### 📊 盘前预测时段（< 09:30）需要的数据：

| 数据类型 | 具体指标 | 用途说明 |
|----------|----------|----------|
| **集合竞价数据** | 竞价成交量、竞价涨跌幅、匹配价格、未匹配买卖量 | 判断开盘方向、资金态度 |
| **盘口挂单** | 实时五档买卖盘（买一-买五、卖一-卖五）、挂单量 | 支撑压力位、买卖力量对比 |
| **资金流向** | 主力资金净流入/流出、大单成交占比 | 机构资金态度、主力动向 |
| **技术形态** | 近期K线形态（日线/60分钟线）、关键均线位置 | 趋势判断、技术支撑压力 |
| **量价关系** | 近期成交量变化、量比、换手率 | 活跃度分析、量价配合 |

**盘前数据获取示例：**
```
# 使用 mx_finance_data 获取盘前数据（推荐，覆盖A港美）

# 或使用 akshare 获取A股盘前数据（备选）
/akshare "东芯股份的集合竞价、盘口挂单、资金流向"
```

#### 📈 盘中/尾盘时段（09:30-15:00）需要的数据：

| 数据类型 | 具体指标 | 用途说明 |
|----------|----------|----------|
| **盘口挂单** | 实时五档买卖盘、挂单量变化、大单挂单 | 支撑压力动态、买卖意愿 |
| **资金流向** | 主力资金净流入/流出、大单动向、散户资金 | 资金流向趋势、主力意图 |
| **技术形态** | 分时图形态、日内高低点、关键价位突破 | 日内趋势、突破信号 |
| **量价关系** | 实时成交量、量比、换手率、量价背离 | 上涨动能、量价配合度 |

### Step 2: 判断交易时段

根据当前时间判断所处时段：

| 时段 | A股时间 | 港股时间 | 分析重点 |
|------|---------|----------|----------|
| 盘前 | < 09:30 | < 09:30 | 预测开盘方向，制定预案 |
| 早盘 | 09:30-11:30 | 09:30-12:00 | 观察开盘表现，判断情绪 |
| 午休 | 11:30-13:00 | - | 复盘上午，准备下午策略 |
| 午盘 | 13:00-14:30 | 13:00-16:00 | 趋势确认，寻找买点 |
| 尾盘 | 14:30-15:00 | 15:30-16:00 | 尾盘机会，仓位调整 |
| 盘后 | > 15:00 | > 16:00 | 次日预判 |

### Step 3: 多模型交叉分析

使用 wei-cross-research 方法，查询 2-3 个模型进行并行分析：

**Answering Models:**
- **kimi-k2.5** - 当前事件、长文本分析、数据解读
- **qwen3.5** - 结构化输出、技术分析、计算
- **minimax-m2.5** - 综合推理、策略制定

**Judge Model:**
- **glm-5-judge** - 综合研判，生成最终交易建议

### Step 4: 输出交易建议

包含以下要素：
- **当前行情概览**
- **市场情绪分析**
- **加仓建议**（是/否/观望）
- **加仓价位区间**
- **仓位建议**
- **止损位设置**
- **风险提示**

## 交易时段详细策略

### 盘前策略 (< 09:30)

**数据来源：** 优先使用 `/mx_finance_data` 获取，备选 `/akshare`
- 集合竞价数据（成交量、匹配价格、未匹配量）
- 盘口挂单（五档买卖盘）
- 资金流向（主力净流入）
- 技术形态（日线趋势、支撑压力位）
- 量价关系（量比、换手率）

**分析重点：**
- 集合竞价量能和价格走势
- 盘口挂单结构和买卖力量对比
- 资金流向显示的主力态度
- 技术形态的支撑位和压力位
- 隔夜美股/欧股走势
- 期货市场表现（A50、恒指期货）
- 重要新闻/政策/公告
- 板块轮动预期

**输出内容：**
- 开盘预测（高开/平开/低开）及概率
- 三种开盘情景的应对预案
- 首批加仓价位和仓位建议
- 关键观察点位（支撑/压力）
- 开盘后观察要点

### 早盘策略 (09:30-11:30)

**数据来源：** 优先使用 `/mx_finance_data` 获取，备选 `/akshare`
- 盘口挂单（五档买卖盘实时变化）
- 资金流向（主力净流入、大单动向）
- 技术形态（分时图形态、日内高低点）
- 量价关系（量比、成交量变化、量价配合）

**分析重点：**
- 开盘半小时表现（定价时段）
- 盘口挂单变化和大单动向
- 资金流向显示的主力态度
- 分时图技术形态
- 成交量对比和量价配合
- 板块资金流向
- 个股相对强度

**输出内容：**
- 开盘质量评估
- 盘口多空力量对比
- 是否追涨/等待回调
- 分批加仓策略
- 上午目标价位
- 关键支撑压力位

### 午休复盘 (11:30-13:00)

**分析重点：**
- 上午走势总结
- 与预测的差异分析
- 下午走势预判
- 资金态度观察

**输出内容：**
- 上午表现评分
- 下午策略调整

## 风险控制原则

1. **仓位控制** - 单只股票不超过总仓位 20%
2. **分批建仓** - 建议分 2-3 批入场
3. **止损纪律** - 跌破关键支撑位果断止损
4. **情绪管理** - 避免追涨杀跌，严格按计划执行
5. **资金配置** - 保留 20-30% 现金应对突发情况

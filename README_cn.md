## wei-cross-research

多 LLM 研究合成，提供高置信度答案。并行查询多个语言模型，将它们的回答合成为单一交叉验证的答案。

### 为什么使用？

当你需要可靠答案而非单一模型的观点时，请使用 **wei-cross-research**。

该技能并行查询多个 LLM，并使用评判模型将它们的回答合成为单一交叉验证的答案。当模型达成一致时，置信度高；当模型存在分歧时，分歧会被呈现而非默默忽略。

**核心优势：**

- **交叉验证减少幻觉** — 多个模型独立回答同一问题；不一致之处会被标记并通过合成解决，相比单一模型查询显著降低 AI 幻觉
- **多样化搜索源** — 不同模型使用不同的搜索索引和数据源；结合使用可提供更广泛的证据覆盖，填补任何单一模型可能遗漏的信息空白
- **更高置信度的答案** — 独立模型之间的一致性表明答案可靠；分歧会被呈现供你审查，而非隐藏

**最适合：**
- 高风险问题，错误答案会带来实际后果
- 单一模型可能存在盲区或偏见的主题
- 受益于多个独立观点的分析
- 需要从多样化来源获得全面证据的研究

> **成本说明：** 该技能每次请求查询 2-3 个模型，token 使用量约为**单一模型查询的 2-3 倍**。权衡在于更高的答案质量和更低的幻觉风险。在准确性重要时使用；避免用于简单或低风险问题。

### 环境要求

- [Bun](https://bun.sh) >= 1.0
- 所用模型提供商的 API 密钥（OpenRouter、Bailian 等）

#### 在 Claude Code / OpenClaw 中使用

```
use cross-research

query="What are the economic impacts of AI agents?"
```

使用可选参数：

```
use cross-research

query="美联储2026年会降息吗？"
maxModels=2
```
